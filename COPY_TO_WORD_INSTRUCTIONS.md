# Quick Guide: Copying to Microsoft Word

## Simple Method (Recommended)

### Step 1: Open the Template File
- Open `QUESTION2_Final_Document.md` in any text editor (Notepad, VS Code, etc.)
- OR open `QUESTION2_Word_Document_Ready.txt` if you prefer plain text

### Step 2: Copy to Word
1. Select all content (Ctrl + A)
2. Copy (Ctrl + C)
3. Open Microsoft Word
4. Create new blank document
5. Paste (Ctrl + V)

### Step 3: Format in Word
1. **Apply Heading Styles:**
   - Select "QUESTION 2.1..." → Home tab → Heading 1
   - Select "Members Table" → Home tab → Heading 2
   - Repeat for all major headings

2. **Format Text:**
   - Body text: Times New Roman or Calibri, 12pt
   - Headings: Make bold, increase size
   - Bullet points will be preserved

3. **Replace Placeholders:**
   - Press Ctrl + H (Find and Replace)
   - Find: `[ENTER YOUR NAME HERE]`
   - Replace with: Your actual name
   - Repeat for other placeholders

### Step 4: Insert Screenshots
1. Find each `[INSERT SCREENSHOT HERE]` placeholder
2. Click on the placeholder text
3. Go to **Insert** tab → **Pictures** → **This Device**
4. Select your screenshot file
5. Click **Insert**
6. Delete the placeholder text
7. Resize image if needed (drag corners)

### Step 5: Final Formatting
- Add page numbers (Insert → Page Number)
- Check margins (Layout → Margins → Normal)
- Review and format captions below screenshots
- Spell check (F7)

### Step 6: Save as PDF
- File → Save As
- Choose location
- File type: PDF
- Click Save

---

## Alternative: Use the Text File

If you prefer plain text formatting:

1. Open `QUESTION2_Word_Document_Ready.txt`
2. Copy all content
3. Paste into Word
4. The formatting will be basic - you'll need to format headings and bullets manually
5. Follow Steps 3-6 above
