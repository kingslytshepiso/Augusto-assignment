# How to Create Your Word Document for Question 2

## Step-by-Step Instructions

### Option 1: Copy from Markdown to Word (Recommended)

1. **Open the File:**
   - Open `QUESTION2_FINAL_DOCUMENT.md` in any text editor
   - Select all content (Ctrl + A)
   - Copy (Ctrl + C)

2. **Paste into Word:**
   - Open Microsoft Word
   - Create a new document
   - Paste (Ctrl + V)
   - Word will preserve most formatting

3. **Format in Word:**
   - Apply heading styles (Heading 1, Heading 2)
   - Adjust font sizes as needed
   - Format bold text appropriately
   - Add page numbers if required

4. **Replace Placeholders:**
   - Find and replace `[ENTER YOUR NAME HERE]` with your actual name
   - Find and replace `[ENTER YOUR STUDENT NUMBER HERE]` with your student number
   - Find and replace `[ENTER DATE HERE]` with the current date

5. **Insert Screenshots:**
   - Place cursor at each `[INSERT SCREENSHOT HERE]` location
   - Go to **Insert** → **Pictures** → **This Device**
   - Select your screenshot image
   - Resize as needed
   - Delete the placeholder text

6. **Final Formatting:**
   - Check all headings are consistent
   - Ensure screenshots are properly sized
   - Add page breaks if needed
   - Check spelling and grammar

7. **Save as PDF:**
   - Go to **File** → **Save As**
   - Choose **PDF** format
   - Save your document

## Screenshot Insertion Instructions

### For Each Screenshot:

1. **Take the Screenshot:**
   - Follow instructions in `QUESTION2_INSTRUCTIONS.md`
   - Save as PNG or JPG file
   - Name clearly: `Q2.3_Relationship.png`, etc.

2. **Insert into Word:**
   - Click where `[INSERT SCREENSHOT HERE]` appears
   - Go to **Insert** tab → **Pictures** → **This Device**
   - Browse and select your screenshot
   - Click **Insert**

3. **Format the Screenshot:**
   - Right-click image → **Wrap Text** → **In Line with Text**
   - Resize by dragging corners (maintain aspect ratio)
   - Ensure image is clear and readable
   - Delete the placeholder text `[INSERT SCREENSHOT HERE]`

4. **Add Caption:**
   - Captions are provided below each screenshot placeholder
   - Ensure caption is directly below the image
   - Format caption as needed (usually smaller font, centered or justified)

## Word Formatting Tips

### Headings:
- **Question Titles:** Use Heading 1 style
- **Sub-questions:** Use Heading 2 style
- **Section Titles:** Use Heading 3 style or Bold

### Fonts:
- **Body Text:** Times New Roman or Calibri, 11-12pt
- **Headings:** Arial or Calibri, 14-16pt
- **Captions:** Times New Roman or Calibri, 10pt, italic

### Spacing:
- Use consistent spacing between sections
- Add page breaks before major sections if needed
- Ensure screenshots have space above and below

### Page Setup:
- Margins: 1 inch (2.54 cm) on all sides
- Page size: A4 or Letter
- Page numbers: Bottom center (optional)

## Troubleshooting

### Screenshots Too Large:
- Right-click image → **Size and Position**
- Set specific width (e.g., 15cm or 6 inches)
- Lock aspect ratio
- Or manually resize by dragging corners

### Screenshots Too Small:
- Increase size by dragging corners
- Or right-click → **Size and Position** → increase percentage

### Text Formatting Issues:
- Select text → apply appropriate styles
- Use Format Painter to copy formatting
- Check if headings are using correct styles

### Screenshots Not Clear:
- Retake screenshots at higher resolution
- Ensure screenshots are saved as PNG (better quality than JPG)
- Check that text in screenshots is readable before inserting
