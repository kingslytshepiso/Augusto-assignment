## Screenshot Tools

### Windows Snipping Tool (Recommended)
- Press: **Win + Shift + S**
- Select area to capture
- Image copied to clipboard
- Paste and save

### Print Screen
- Press: **Print Screen** key
- Paste into Paint or image editor
- Crop to show only relevant area
- Save

### File Format
- Save as: **PNG** (best quality) or **JPG**
- Resolution: Minimum 800px width recommended
- Quality: Text must be clear and readable

## Including Screenshots in PDF

### Best Practices:
1. Insert screenshots after written answers
2. Add captions below each screenshot:
   - "Figure 1: Relationship Diagram"
   - "Figure 2: Query Result"
   - "Figure 3: Tabular Form"
   - "Figure 4: Report"
3. Ensure screenshots are properly sized
   - Not too small (unreadable)
   - Not too large (takes up whole page)
4. Check readability
   - Text must be clear
   - No blurriness
   - Proper contrast

